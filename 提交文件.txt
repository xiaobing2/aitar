【作品说明】

一、项目概述
Glass Memo 是一款基于边缘计算和AI技术的智能备忘录应用，采用3D玻璃卡片式设计，支持QQ群消息智能识别并自动转换为任务。通过阿里云通义千问AI能力，实现智能任务提取、计划制定等功能，让任务管理变得更加智能和高效。

二、核心功能
1. 3D玻璃卡片式UI：采用Glassmorphism设计风格，结合Three.js实现3D交互背景，提供沉浸式视觉体验
2. AI智能任务识别：集成阿里云通义千问API，自动识别QQ群消息中的任务信息，提取标题、详情和截止时间
3. AI计划制定：通过自然语言描述，AI自动生成结构化计划，包含多个子任务和完成步骤
4. 群聊任务同步：实时接收QQ群消息，自动识别任务并同步到任务列表
5. 边缘计算架构：基于阿里云ESA边缘函数，实现低延迟的消息处理和API代理

三、技术实现
1. 前端技术栈：Vue.js 2.6 + Vuex + Vue Router + Three.js
2. 边缘计算：阿里云ESA边缘函数处理QQ Webhook和API代理
3. AI能力：阿里云通义千问API实现智能任务识别和计划生成
4. 存储方案：localStorage + IndexedDB本地存储

四、应用价值（实用性）
1. 工作场景：团队协作中，群聊消息中的任务要求经常被遗漏。Glass Memo可以自动识别并提取任务，确保重要事项不被遗忘
2. 个人管理：通过AI能力，将自然语言快速转换为结构化任务和计划，提高个人效率
3. 零配置使用：部署即用，无需复杂配置，适合个人和团队快速上手

五、创意亮点（创意性）
1. 视觉创新：3D玻璃卡片设计结合动态背景，打破传统备忘录的单调界面
2. 交互创新：卡片悬浮、3D旋转等交互效果，让任务管理变得有趣
3. AI创新：将AI能力深度融入任务管理流程，实现真正的智能助手

六、技术深度（技术探索）
1. 边缘计算：充分利用ESA边缘函数，实现低延迟的消息处理和API代理，解决CORS问题
2. AI集成：深度集成阿里云通义千问API，实现智能任务识别和计划生成，支持自然语言理解
3. 3D渲染：使用Three.js实现高性能3D场景渲染，优化性能表现，提供流畅的交互体验
4. 前端架构：Vue.js + Vuex状态管理，组件化设计，代码结构清晰，易于维护和扩展

七、ESA产品使用情况
1. ESA Pages：部署前端应用，提供全球加速和CDN服务
2. 边缘函数：处理QQ Webhook回调、API代理、消息存储
3. 边缘存储：使用边缘KV存储消息数据（可扩展）
4. 边缘缓存：利用ESA的缓存能力提升API响应速度

八、使用说明
1. 部署前端到ESA Pages
2. 部署边缘函数到ESA边缘函数平台
3. 配置QQ机器人Webhook地址
4. 在设置页面配置阿里云API Key
5. 在QQ群中@机器人发送任务消息即可自动识别

九、项目特色
- 原创设计：3D玻璃卡片式UI设计，视觉效果美观
- 技术先进：充分利用边缘计算和AI技术
- 实用性强：解决实际工作场景中的任务管理问题
