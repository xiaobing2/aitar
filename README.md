## 📖 项目简介

Glass Memo 是一款基于边缘计算和AI技术的智能备忘录应用，采用3D玻璃卡片式设计，支持QQ群消息智能识别并自动转换为任务。通过阿里云通义千问AI能力，实现智能任务提取、计划制定等功能，让任务管理变得更加智能和高效。

### ✨ 核心特性

- **🎨 3D玻璃卡片式UI**：采用Glassmorphism设计风格，结合Three.js实现3D交互背景，提供沉浸式视觉体验
- **🤖 AI智能任务识别**：集成阿里云通义千问API，自动识别QQ群消息中的任务信息，提取标题、详情和截止时间
- **📋 AI计划制定**：通过自然语言描述，AI自动生成结构化计划，包含多个子任务和完成步骤
- **💬 群聊任务同步**：实时接收QQ群消息，自动识别任务并同步到任务列表
- **⚡ 边缘计算架构**：基于阿里云ESA边缘函数，实现低延迟的消息处理和API代理
- **📱 响应式设计**：完美适配PC、平板和移动设备

## 🛠️ 技术架构

### 前端技术栈
- **框架**：Vue.js 2.6 + Vuex + Vue Router
- **3D渲染**：Three.js
- **UI设计**：Glassmorphism（玻璃态设计）
- **状态管理**：Vuex
- **HTTP客户端**：Axios
- **存储**：localStorage + IndexedDB

### 边缘计算
- **平台**：阿里云ESA边缘函数
- **功能**：
  - QQ Webhook接收和验证
  - 阿里云API代理（解决CORS问题）
  - 消息存储和查询
  - Ed25519签名验证

### AI能力
- **服务商**：阿里云通义千问
- **功能**：
  - 智能任务识别（从消息中提取任务信息）
  - 计划制定（生成结构化计划）
  - 自然语言理解

## 🚀 快速开始

### 环境要求
- Node.js >= 14.0
- npm >= 6.0

### 安装依赖
```bash
npm install
```

### 本地开发

1. **启动边缘函数服务器**
```bash
cd functions
npm install
node server.js
```

2. **启动前端开发服务器**
```bash
npm run serve
```

3. **访问应用**
打开浏览器访问：`http://localhost:8080`

### 配置说明

#### 边缘函数配置
在 `functions/.env` 文件中配置：
```env
QQ_SECRET=your_qq_bot_secret_here
NODE_ENV=development
```

#### 前端配置
在设置页面配置阿里云API Key，用于AI功能。

参考 `functions/README.md` 中的详细部署指南。
